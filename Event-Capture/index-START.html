//Propagation, Bubbling, event capturing and once //Bubbling - where the browser will figure what you clicked on but will trigger clicks up, will ripple up to the top of the document and trigger clicks on those elements as well //Capture - the function will run on the way down instead of running on the way up like bubbling //e.stopPropagation - This stops the event from bubbling up or you can run it in conjuction with capture so that it stops at the top element //Once - listens for a click once and then unbinds itself so that there is no future clicks on it. Good feature for store check outs
